# Contributing to SuiClient Skill

Thank you for your interest in contributing to the SuiClient Skill! This document provides guidelines and instructions for contributing.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/sui-client-skill.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Code Style

- Use TypeScript for all new code
- Follow the existing code style and patterns
- Include JSDoc comments for public functions
- Write tests for new functionality

## Testing

Run tests with:
```bash
npm test
```

## Building

Build the project with:
```bash
npm run build
```

## Pull Request Process

1. Ensure all tests pass
2. Update documentation if needed
3. Add a clear description of changes
4. Reference any related issues

## Skill Structure

- `src/` - Main skill implementation
- `types/` - TypeScript type definitions
- `examples/` - Usage examples
- `templates/` - Code templates
- `tests/` - Test files

## Questions?

Feel free to open an issue for any questions or discussions.

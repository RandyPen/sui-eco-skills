# Basic Usage Example

This example demonstrates the basic usage of Cetus Aggregator SDK for executing optimized swaps.

## Setup

1. Copy `.env.example` to `.env` and fill in your values
2. Install dependencies: `bun install`
3. Run the example: `bun run dev`

## Features Demonstrated

- SDK initialization with environment variables
- Query swap with profit conditions
- Simple swap execution
- Error handling and memory cleanup

## Usage

```bash
bun run dev
```

This will execute example swaps using the Cetus Aggregator SDK.
